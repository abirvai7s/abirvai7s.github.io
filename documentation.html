Project Overview

7S WORLD 2.0 is an advanced Discord bot designed for high-end server security (Antinuke) and professional community management. Powered by a persistent MongoDB database, your configurations remain safe even after restarts. It features a scalable Three-Tier License System (Basic, Standard, Premium) ideal for commercial use and community growth.

Setup Documentation

Step 1: Environment Configuration

Ensure you have Node.js v18+ installed. Create a .env file in your root directory and add the following:

TOKEN=YOUR_BOT_TOKEN
MONGO_URI=mongodb+srv://user:pass@cluster.mongodb.net/7S_WORLD
OWNER_ID=YOUR_DISCORD_ID

Step 2: Install Dependencies

Run the following command in your terminal or double-click install.bat:

npm install discord.js mongoose dotenv colors ms

Step 3: Launch the Bot

node src/index.js

Essential Commands

Command Description Required Tier
!prefix [new] Change the bot prefix Basic
!config link on/off Toggle link protection Standard
!antinuke on/off Toggle Antinuke Guard Premium
!whitelist add @user Add a trusted user Premium
!set-license [id] [tier] Update Server Tier Owner Only

Critical Security Notice

For the Antinuke punishments (Ban/Kick) to work effectively, the 7S WORLD 2.0 bot role must be positioned ABOVE all administrator and moderator roles in your Discord Server Settings. If the role is lower, the bot will not have the hierarchy to punish malicious staff members.
