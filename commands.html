Command List

Complete directory of 7S WORLD 2.0 executable commands.

Basic Standard Premium
Administration
!prefix Basic
Changes the bot prefix for your server.
!prefix <new_prefix>
!welcome Basic
Sets the channel for automated welcome messages.
!welcome #channel
Modules
!ticket Basic
Sends the interactive button panel for support tickets.
!ticket
!gstart Standard
Launches a giveaway with automatic winner selection.
!gstart <time> <prize>
!config Standard
Enables or disables Anti-Link and Anti-Spam modules.
!config <link/spam> <on/off>
Security (Antinuke)
!antinuke Premium
Main switch for the Antinuke Core Guard.
!antinuke <on/off>
!whitelist Premium
Manages users who are exempt from all security triggers.
!whitelist <add/remove> @user
!lockdown Premium
Locks the entire server by disabling permissions for @everyone.
!lockdown <on/off>
!antibot Premium
Toggles unauthorized bot protection.
!antibot <on/off>
